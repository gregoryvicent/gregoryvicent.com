## Sobre este proyecto

gregoryvicent.com nace de la necesidad que me surgió de tener una plataforma propia donde poder mostrar tanto mis trabajos como programador así como tener un lugar en donde compartir mis ideas y pensamientos. Con esto en mente desarrollo este pequeños proyecto que servirá de sitio de reunión para las personas que compartan mis intereses o que deseen saber un poco más de mi como profesional y del contenido que creo.

## Tecnologías Usadas

- *Next.js*
- *JSX*
- *CSS*
